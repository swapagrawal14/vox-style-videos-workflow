• Typography

• Texture
• Grain

• Negative space
• Editorial metaphors
• Motion language

Never output arrays if the user wants individual prompts.
Each prompt is standalone JSON.

title, scene, purpose, composition, camera, subject, background, style, lighting, color_palette, prompt, negative_prompt.
